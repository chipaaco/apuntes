# Cinemática

Estudio del movimiento sin considerar las causas que lo producen.

1. **Posición** y desplazamiento
2. **Velocidad** media e instantánea
3. **Aceleración**
4. **Movimiento rectilíneo uniforme** (MRU)

## Conceptos clave

- Trayectoria
- Sistema de referencia
- Ecuaciones horarias del movimiento
