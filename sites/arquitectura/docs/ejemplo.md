# Modelo Von Neumann

El modelo Von Neumann describe una arquitectura de computador que consiste en:

1. **Unidad de procesamiento** (ALU + registros)
2. **Unidad de control**
3. **Memoria** (almacena datos e instrucciones)
4. **Dispositivos de entrada/salida**

## Características principales

- Programa almacenado en memoria
- Ejecución secuencial de instrucciones
- Bus único para datos e instrucciones (cuello de botella de Von Neumann)
